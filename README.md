## About The Project

This is a Stock Alert News project. The program will send you the latest news articles about a company, depending on the company stocks you are tracking.
It will send the news alerts based on the difference in stock closing prices of the previous two days. 


<!-- USAGE EXAMPLES -->
## Usage

The program can be used to keep track of stock prices and to recieve news alerts about the tracked stock prices.
